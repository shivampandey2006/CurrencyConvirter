# CurrencyConvirter
 Features
- Converts USD → INR
- Converts INR → USD
- Works completely offline
- Clean and easy UI
- Beginner-friendly project

Technologies Used
- HTML
- JavaScript

 How to Run
1. Download or copy the project code.
2. Save file as: `converter.html`
3. Open the file in any browser
4. Enter amount → select conversion → click Convert

 Conversion Rates Used
- 1 USD = 83 INR  
- 1 INR = 0.012 USD  
 Output Example
- Enter amount  
- Select direction (USD → INR / INR → USD)  
- Click convert  
- See the converted value!
